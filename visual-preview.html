‚ú® Key UI Elements You Can Customize

Colors & Branding

  • Button colors (currently blue gradient)
  • Badge colors for To/CC/BCC
  • Background gradients
  • Header text and icons
  • Statistics display colors

Layout & Spacing

  • Contact card design
  • Statistics grid layout (3 columns)
  • Padding and margins
  • Font sizes and weights
  • Border radius for rounded corners

Functionality

  • Email scan limit (default 500)
  • Number of emails shown per contact
  • Sort order (by count, date, etc.)
  • Which folders to scan
  • Date range filtering

Text & Labels

  • Add-in title and subtitle
  • Button labels
  • Status messages
  • Empty state messages
  • Statistics labels
üí° How to Request Changes
Now that you've seen how it looks, you can request any changes such as:
  • Design changes: "Change the blue buttons to green" or "Make the cards have more shadow"
  • Layout changes: "Show 4 statistics instead of 3" or "Make contact cards wider"
  • Functionality changes: "Scan 1000 emails instead of 500" or "Add a date filter"
  • Text changes: "Change the title to 'Contact Manager'" or "Add company logo"
  • Feature additions: "Add export to Excel" or "Show email subjects in list"
